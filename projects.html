My personal projects...

Here you can find some of the projects that I have worked on. These projects have helped me learn and improve while also allowing me to explore my creativity. I am excited to continue building and expanding my portfolio as new ideas come to mind in the future!

C++

ATM Application

A console application that simulates an ATM, allowing users to perform basic banking operations.
The purpose of this project was to practice C++ programming and also familiarize myself with SQL (for this I used PostgreSQL).

C++

Animal Farm

This is a simple program made to simulate the management of animals on a farm.
You can add, delete animals from the system or modify certain details about each entry.

C++

Managing Store Inventory

Simple project meant to familiarize myself and better understand the concept of arrays in a fun way.
Upon running the program, users are prompted to search, sort or calculate the total price of selected products.

Python

Veterinary Clinic Management System(WIP)

An application meant for the internal management of a veterinary clinic, allowing staff to manage patients, records, appointments and more.
This project was built using Django and Python with a PostgreSQL database and a touch of front-end development.

Python

Mini Target Practice Game

A simple game built with Pygame, where the player can practice their aim by shooting targets appearing on screen.
Inspired by classic practice ranges in video games, this project helped me practice various programming concepts in a fun and engaging way.

Python

My Gaming Diary(WIP)

Gaming Diary's dreams and aspirations are to keep a detailed track of Steam games you've played.
My passion for gaming and wanting to keep a detailed track of gaming sessions, inspired me to create a diary which does just that for you.

PHP

ATM Application

A PHP version of the C++ "ATM Application"
This project allowed me to explore web development using PHP and PostgreSQL, recreating the "ATM Application" and giving it a web-based interface.
